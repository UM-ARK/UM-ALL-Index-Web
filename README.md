## 多語言修改方式
- 只需要修改 `data\languge.json` 就可以。
- 主key對應html標籤的name
- en是必須有的，如果其他語言留空，就會自動使用en的內容

## 修改用户协议
- 只需要修改 `data\useragreement.md` 就可以。
